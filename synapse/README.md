# Synapse Simulator - Development Notes

## Versioni

### v1 - Animation-based (synapse-simulator-v1-animation.html)
Prima versione con animazione Canvas della cascata sinaptica.

**Problema identificato**: L'animazione è sempre identica per tutti gli scenari, cambia solo il colore. Poco engaging per ripetizioni.

**Vedi**: `OPTION-A-differentiated-animations.md` per proposta di evoluzione con animazioni differenziate per scenario.

### v2 - Infografica interattiva (in sviluppo)
Approccio diverso: invece di animazione ripetitiva, infografica con illustrazioni specifiche per ogni scenario e elementi interattivi.

## Scenari

### Stimoli naturali
- Dolore (Glutammato)
- Piacere (Dopamina)  
- Paura (Noradrenalina)

### Sostanze quotidiane
- Caffè (blocco Adenosina)
- Camomilla (potenziamento GABA)
- Vino (potenziamento GABA + riduzione Glutammato)
- Cioccolato (Endorfine)

## Note pedagogiche

- Includere sempre nota sulla dose per le sostanze
- Collegare meccanismo molecolare → effetto percepito (macro)
- Target: pubblico generale senza background scientifico
