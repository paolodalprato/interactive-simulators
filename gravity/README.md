### Try These Scenarios

**Rapid cluster formation**:

- G = 300-400
- Particles = 100-150
- Initial velocity = 0.5

**Slow aggregation**:

- G = 20-30
- Particles = 50
- Initial velocity = 1.0

**Few trackable particles**:

- G = 100
- Particles = 20-30
- Initial velocity = 0.3
